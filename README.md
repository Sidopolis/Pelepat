# 🚀 Pelepat

> Smart automation toolkit for developer workflows.

Pelepat is a Python-based automation engine that streamlines development workflows — from issue tracking and code generation to automated reviews and releases.

## ✨ Features

- 🐛 **Automated Issue Management** — Create, track, and close issues programmatically
- 🔀 **Smart PR Workflows** — Auto-generate code fixes and open pull requests
- 👁️ **Code Review Automation** — Automated peer reviews with approval + merge
- 📦 **Release Management** — Semantic versioning with automated changelogs
- 🤝 **Multi-contributor Support** — Co-authored commits for team collaboration

## 🛠️ Quick Start

```bash
# Clone the repository
git clone https://github.com/Sidopolis/Pelepat.git
cd Pelepat

# Install dependencies
pip install -r requirements.txt

# Run the toolkit
python main.py
```
